# Mission: AI Engineering

## Why
To become a genuinely efficient AI engineer — someone who can design, build, and debug complex agent systems, not just call APIs. Understanding how agents think, how to orchestrate them, and how to build reliable systems on top of non-deterministic models.

## Success looks like
- Design and implement a multi-agent system from scratch
- Choose the right orchestration pattern for a given problem (ReAct, plan-and-execute, tool-use, etc.)
- Build agents with memory, tool use, and reliable output
- Read and understand the agent frameworks (Claude API, OpenAI Assistants, LangGraph, etc.) at a deep level
- Write production-quality agentic code

## Constraints
- Experienced software engineer (Rust/C background)
- Prefer depth and first-principles over framework tutorials
- Code examples in Python where idiomatic for this domain; Rust where it makes sense

## Out of scope
- Basic LLM API usage (assumed known)
- MLOps / training infrastructure
