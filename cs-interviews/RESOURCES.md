# CS Interview Prep Resources

## Knowledge

- [Tech Interview Handbook (techinterviewhandbook.org)](https://www.techinterviewhandbook.org/)
  Comprehensive, free, community-maintained guide covering patterns, study plans, and cheat sheets. Use for: topic roadmap and algorithm study sheets.

- [Grokking the Coding Interview Patterns (educative.io)](https://www.educative.io/courses/grokking-coding-interview)
  Pattern-based approach — teaches the 16 core patterns that cover ~80% of interview problems. Use for: pattern recognition training.

- [The Rust Book (doc.rust-lang.org)](https://doc.rust-lang.org/book/)
  Official Rust reference. Use for: verifying Rust syntax and idioms in code examples.

- [Karat Engineering Interview Trends 2026 (karat.com)](https://karat.com/engineering-interview-trends-2026/)
  Data-driven report on what companies actually ask in 2026. Use for: calibrating which topics to prioritise.

- [NuCamp — Data Structures & Algorithms in 2026](https://www.nucamp.co/blog/data-structures-and-algorithms-in-2026-what-backend-developers-actually-need-to-know-and-what-ai-can)
  Current overview of what's changed and what AI can't replace. Use for: framing the "why bother" question.

- [LeetCode Patterns That Still Matter in 2026 (interviewaibox.co)](https://interviewaibox.co/en/blog/leetcode-patterns-that-still-matter-2026)
  Curated list of patterns still actively tested. Use for: prioritising which patterns to study first.

## Wisdom (Communities)

- [r/cscareerquestions](https://www.reddit.com/r/cscareerquestions/)
  High-volume, well-moderated. Use for: real candidate reports on recent interview experiences at specific companies.

- [r/rust](https://www.reddit.com/r/rust/)
  Friendly Rust community. Use for: idiomatic Rust questions on data structure implementations.

- [Blind (teamblind.com)](https://www.teamblind.com/)
  Anonymous forum used heavily by FAANG engineers. Use for: company-specific interview intel (take with a grain of salt).
