# Baseline: Technical Background, Debate-Driven Motivation

User has no prior formal logic training but has a strong technical/CS background (comfortable with formal notation, proof-like reasoning, Rust/C). Motivation is following philosophical and public debates rigorously — evaluating argument structure independent of topic. Explicitly does not want the course centered on any single application (e.g. religion), even though that's part of the motivation.

**Implications:** Skip hand-holding on symbolic notation — introduce it directly and move quickly. Prioritize real argument examples (from any domain) over abstract exercises. Keep lessons topic-neutral; let the user apply the tools themselves.
