Proofs in Predicate Logic

Six lessons ago, the Socrates argument was just prose. Lesson 7 gave it precise symbols. This lesson gives you the four rules that let you actually prove it — moving between "true for everyone" and "true for this one specific thing," rigorously, with the same line-by-line discipline as lesson 5's natural deduction.

Quick recall

From lesson 7, the Socrates argument is fully symbolized: ∀x (Human(x) → Mortal(x)), Human(Socrates) ⊢ Mortal(Socrates). This lesson proves it.

Universal Instantiation (UI)

∀x P(x) ⊢ P(a) — if something is true for everyone, it's true for any specific individual a you name. Always safe, no restrictions: universal claims apply to every member of the domain, so picking out any one of them is never a problem.

Existential Generalization (EG)

P(a) ⊢ ∃x P(x) — if a specific individual has a property, then something has that property. Also always safe: you're not claiming which thing has it beyond what you already showed, just that at least one thing does.

Existential Instantiation (EI)

∃x P(x) ⊢ P(c) — if something has property P, give that thing a temporary name c and reason about it. Restriction: c must be a fresh name — not already used elsewhere in the proof for something else. ∃x P(x) only tells you something satisfies P; it doesn't tell you it's the same thing as whatever you already called Socrates. Reusing an old name would be smuggling in an assumption you haven't earned.

Universal Generalization (UG)

P(a) ⊢ ∀x P(x) — if you can prove P holds for a, and a was never anything more than an arbitrary stand-in for "any member of the domain" — never a specific named individual with special properties, never something obtained via EI — then P holds for everyone.

Why UG and EI need restrictions, and UI/EG don't UI and EG only ever narrow a claim — from "everyone" to "one specific case," or from "one specific case" to "at least one exists." Narrowing is always safe. UG and EI both try to widen a claim — from one case to "everyone," or from "something exists" to "here's a name for it." Widening is exactly where mistakes creep in: without the restriction, you could prove Human(Socrates) ⊢ ∀x Human(x) — "Socrates is human, therefore everyone is human" — which is obviously not licensed by anything you actually established. The restriction is what stops that.

Closing the loop: proving the Socrates argument

Given: ∀x (Human(x) → Mortal(x)), Human(Socrates). Prove: Mortal(Socrates).

1∀x (Human(x) → Mortal(x))premise
2Human(Socrates)premise
3Human(Socrates) → Mortal(Socrates)1, Universal Instantiation
4Mortal(Socrates)2, 3, Modus Ponens

That's it — the argument that opened this entire topic, six lessons ago, now proven with total rigor: one quantifier rule (UI) to pull the specific instance out of the universal claim, then Modus Ponens to finish it, exactly the way it would have worked in pure propositional logic once the universal was instantiated down to a plain conditional.

A second worked proof — combining all four rules

Given: ∀x (Human(x) → Mortal(x)), ∃x Human(x). Prove: ∃x Mortal(x).

1∀x (Human(x) → Mortal(x))premise
2∃x Human(x)premise
3Human(c)2, Existential Instantiation (c fresh)
4Human(c) → Mortal(c)1, Universal Instantiation
5Mortal(c)3, 4, Modus Ponens
6∃x Mortal(x)5, Existential Generalization

Notice the pattern: EI first (to get a concrete name to work with), then UI and the propositional rules do the middle work, then EG at the end to generalize back to an existence claim. This EI-early, EG-late shape is extremely common — it's usually the first thing to try when a proof mixes ∀ and ∃ premises.
