Syntax vs. semantics

SymbolMeaning
⊢"Proves" — a proof exists using the rules (syntactic)
⊨"Entails" — true in every model / no counterexample row (semantic)

Soundness & completeness (of a system)

PropertyDefinition
Soundness⊢ → ⊨ — everything provable is actually valid
Completeness⊨ → ⊢ — everything valid is actually provable
Not the same "sound" as lesson 1 Lesson 1's soundness = one argument, valid + true premises. This soundness = a whole proof system, its rules never derive an invalid argument. Same word, different level.

Decidability

Propositional logic: decidable (truth tables always terminate). Full FOL: complete but not decidable — a proof is guaranteed to exist for every valid argument, but no algorithm always terminates to tell you valid vs. invalid.

Modal operators

SymbolMeaning
□P"necessarily P" — true in every possible world
◇P"possibly P" — true in at least one possible world

Modal duality

OriginalEquivalent
¬□P◇¬P
¬◇P□¬P
The modal scope fallacy □(P → Q) ("the conditional is necessarily true") ≠ P → □Q ("P being true forces Q to be necessary"). Proof they differ: □(Rain→Rain) is true (tautology), but Rain→□Rain is false (raining is contingent, not necessary).

Gödel's theorems

TheoremSays
Completeness (1929)First-order logic itself: every valid argument (⊨) is provable (⊢). About logic in general.
1st Incompleteness (1931)Any consistent system strong enough for arithmetic has true statements it can't prove.
2nd Incompleteness (1931)Such a system can't prove its own consistency from within.
Before citing Gödel outside mathematics Ask: which formal system, which statement, what exactly is claimed unprovable? No answer means it's rhetoric, not the actual theorem at work. The 1929 and 1931 results are not contradictory — they're about different objects (logic in general vs. a specific rich theory).

The proof machinery (mastery phase)

PieceWhat it does
Gödel numberingEncodes every symbol/formula/proof as a unique natural number (prime-power scheme) — syntax becomes arithmetic
Prov(y)∃x. Proof(x, y) — "y is provable," an ordinary arithmetic formula, taken via representability
Diagonal lemmaFor any ψ(x), builds a sentence φ with ⊢ φ ↔ ψ(⌜φ⌝) — the quine-like self-reference engine
GDiagonal lemma applied to ψ(x) = ¬Prov(x): G ↔ ¬Prov(⌜G⌝) — "I am not provable," built, not asserted
