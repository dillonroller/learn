Rules of Inference Cheat Sheet

The compressed version of lesson 5. Every rule here is itself a valid argument form, checkable by truth table.

Core rules

"⊢" (turnstile) means "therefore" — everything left of it must already be established for the rule to apply.

RulePattern
Modus Ponens (MP)P → Q, P ⊢ Q
Modus Tollens (MT)P → Q, ¬Q ⊢ ¬P
Hypothetical Syllogism (HS)P → Q, Q → R ⊢ P → R
Disjunctive Syllogism (DS)P ∨ Q, ¬P ⊢ Q
Conjunction (Conj)P, Q ⊢ P ∧ Q
Simplification (Simp)P ∧ Q ⊢ P
Addition (Add)P ⊢ P ∨ Q

Invalid look-alikes — do not confuse these

Invalid patternLooks like
Denying the Antecedent
P→Q, ¬P ⊢ ¬QModus Tollens, but ¬P and ¬Q are swapped
Affirming a Disjunct
P∨Q, P ⊢ ¬QDisjunctive Syllogism, but ∨ is inclusive — both can be true
Affirming the Consequent
P→Q, Q ⊢ PModus Ponens, run backwards

Proof by contradiction

Assume the opposite of what you want to prove, derive a contradiction, conclude the assumption was false.

1A → Bpremise
2A → ¬Bpremise
3Aassumption
4B1,3 MP
5¬B2,3 MP
6B ∧ ¬B4,5 Conj — contradiction
7¬A3–6, reductio
