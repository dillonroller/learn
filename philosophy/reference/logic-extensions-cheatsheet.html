Logic Extensions Cheat Sheet

Deontic, epistemic, and transcendental — three ways of pushing formal reasoning past what classical propositional/predicate logic covers.

Deontic logic — obligation & permission

OperatorMeaning
OpObligatory — true in every deontically ideal world. (≈ □p, normative reading)
PpPermissible — true in some deontically ideal world. (≈ ◇p, normative reading)
FpForbidden — defined as O¬p.
DualityPp ≡ ¬O¬p  ·  Op → Pp (obligatory ⟹ permissible)
The is-ought gap No purely descriptive ("is") premises validly entail a prescriptive ("ought") conclusion — a normative premise must bridge the two, or the argument is missing an implicit premise.

Epistemic logic — knowledge & belief

OperatorMeaning
KpKnows that p — true in every world consistent with what's known. Factive: Kp → p.
BpBelieves that p — same structure as K, but not factive (Bp → p is invalid).
IntrospectionPositive: Kp → KKp. Negative: ¬Kp → K¬Kp. Both stronger, more contested than factivity.

Transcendental arguments

PieceRole
YThe undeniable starting fact (e.g. "we have coherent experience of an objective world")
XThe claimed necessary precondition for Y being possible at all (e.g. causality)
Uniqueness objectionIs X really the ONLY precondition for Y, or could a rival X′ also explain it?
