# Rationality split off from Logic, explicitly for relationships and everyday life

The user, after finishing the Logic curriculum, asked what would help them be "more logical with human relationships as well as with things in our life." This is a different goal than Logic's — Logic evaluates argument structure and spots fallacies; this is about catching one's own biased reasoning in real time and reasoning well under uncertainty, especially with people you're close to. This sets the zone of proximal development for this whole topic: every lesson should be graded on "does this help in an actual disagreement with someone I care about," not just "is this technically correct rationality content." The user confirmed splitting this into its own topic (separate from [[../philosophy/learning-records/0001-arrived-via-logic-extensions.md|Philosophy's more formal Logic Extensions]]) rather than folding it into either Logic or Philosophy.
